# 🚀 LeadFlow AI

LeadFlow AI is an AI-powered Event Lead Management System built using Flask, SQLite, Bootstrap, and Python.

Demo link: https://leadflow-ai-qojv.onrender.com

## Features

- 📝 Lead Registration
- 🤖 AI Priority Scoring
- 🎁 AI Package Recommendation
- 💬 AI Customer Response Generation
- 📊 Dashboard with KPIs
- 📈 Analytics Dashboard
- 🥧 Event Distribution Pie Chart
- 📉 Priority Distribution Bar Chart
- 📁 Export to CSV
- 📗 Export to Excel
- ✏️ Update Lead Status
- 🗑️ Delete Leads

## Technologies Used

- Python
- Flask
- SQLite
- Bootstrap 5
- Matplotlib
- OpenPyXL

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```
http://127.0.0.1:5000
```
